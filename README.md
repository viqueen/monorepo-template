## development environment

- Docker
- Node.js (nvm)
- Yarn
- Golang

## dev setup

- install dependencies

```bash
yarn
```

- schema: build required tools

```bash
yarn schema:tools
```

- schema: codegen

```bash
yarn schema:codegen
```
